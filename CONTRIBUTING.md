# Contributing to AI Quiz Master

Thank you for your interest in contributing! This document provides guidelines and instructions for contributing to the AI Quiz Master project.

## Code of Conduct

- Be respectful and inclusive
- Focus on the content and ideas, not individuals
- Help others learn and grow

## How to Contribute

### Adding Quizzes

1. **Choose a Category**: Determine if your quiz belongs in:
   - `quizzes/school/` - K-12 grade-level content
   - `quizzes/professional/` - Topic or skill-based content
   - `quizzes/master-class/` - Comprehensive mastery content

2. **Follow the Schema**: Use the standardized markdown format defined in [docs/QUIZ_SCHEMA.md](docs/QUIZ_SCHEMA.md)

3. **Quality Checklist**:
   - Questions are clear and unambiguous
   - Correct answers are accurate
   - Difficulty level is appropriate for target audience
   - Content is factually correct and up-to-date
   - No copyright violations

4. **Submit a Pull Request**:
   - Create a fork of the repository
   - Add your quiz file(s)
   - Submit a PR with a clear description
   - Respond to review feedback

### Improving Documentation

- Fix typos and clarify instructions
- Update outdated information
- Improve schema documentation

### Reporting Issues

- Check if the issue already exists
- Provide clear, specific details
- Include examples if applicable

## Quiz Submission Guidelines

### File Naming
- Use kebab-case: `nlp-fundamentals-beginner.md`
- Include topic and difficulty level where applicable

### Content Requirements
- Questions should test understanding, not just memorization
- Provide detailed explanations for correct answers
- Ensure accuracy and currency
- Appropriate for the target audience (grade level or skill level)

## Review Process

All submissions will be reviewed for:
- Accuracy and factual correctness
- Clarity and appropriate difficulty
- Format compliance with schema
- Alignment with project goals
- Quality and educational value

## Questions?

Feel free to open an issue to discuss ideas or ask questions before contributing!
